Добро пожаловать!

Это учебный проект к выпускной квалификационной работы на тему "Разраотка информационной системы пищеблока Адамовской районной больницы".

Данный проект представляет собой веб-приложение построенное с ипользованием языка программирования Java и фреймворка Spring.

Для навигации по веб-приложению воспользуйтесь навигационным меню сверху.

Автор проекта: Сидорин Александр
